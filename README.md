# 🚧 RoadScan — AI Road Damage Detection System

RoadScan is a Streamlit-based AI application that detects and classifies road surface damage from images using a Convolutional Neural Network (CNN). It provides real-time predictions with confidence scoring and a modern, luxury-style UI.

---

## ✨ Features

- 📸 Upload road surface images (JPG, JPEG, PNG)
- 🧠 CNN-based deep learning classification
- 📊 Confidence score visualization with progress bar
- 🎯 Severity indicator (High / Moderate / Low Confidence)
- 🎨 Modern luxury minimalist UI design
- ⚡ Real-time inference with TensorFlow
- 🖥️ Fully interactive Streamlit web app

---


## 🧠 Model Details

- Framework: TensorFlow / Keras
- Model Type: Convolutional Neural Network (CNN)
- Input Size: 128 × 128 pixels
- Output: Multi-class road damage classification
- Activation: Softmax
- Labels: Stored in `labels.json`
